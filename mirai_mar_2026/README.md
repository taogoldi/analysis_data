# Mirai Analysis Bundle (Mar 2026)

This folder contains reproducible analysis artifacts for the Mirai-like Stage1 sample discussed in the public write-up.

Contents:
- `notebooks/`: Jupyter walkthrough
- `scripts/`: extraction and triage scripts
- `ida/`: IDAPython helper scripts
- `reports/`: generated JSON/disassembly/static reports
- `detection/`: YARA rules for this sample cluster
- `images/`: screenshots used in the report
- `docs/`: workflow and comparison notes
