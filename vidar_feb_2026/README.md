# vidar_feb_2026

Public analysis bundle for the `22.exe` loader/stager write-up.

## Contents
- `docs/`: workflow and flowchart sources.
- `scripts/`: extraction/evasion/IOC/similarity scripts.
- `notebooks/`: walkthrough notebook (output-cleared).
- `ida/`: IDAPython annotation helpers.
- `reports/`: generated JSON reports, disassembly dumps, image evidence.
- `input/`: hash metadata only (sample not redistributed).

## Notes
- This dataset is intended for reproducible reverse-engineering workflows.
- Malware binaries are not redistributed here.
